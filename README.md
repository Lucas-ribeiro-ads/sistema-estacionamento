# Sistema de Gerenciamento de Estacionamento

Projeto de sistema para controle de entrada e saída de veículos desenvolvido com tecnologias web fundamentais.

## Funcionalidades
- **Registro de Entrada**: Cadastro de placa, modelo e condutor.
- **Validação**: Verificação de campos obrigatórios e bloqueio de placas duplicadas.
- **Automação**: Registro automático de data e hora no momento da entrada.
- **Controle de Ocupação**: Contador de vagas em tempo real via manipulação de DOM.
- **Registro de Saída**: Remoção dinâmica de veículos da listagem.

## Tecnologias
- HTML5
- CSS3 (Grid Layout)
- JavaScript
